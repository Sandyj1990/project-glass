# Trend-to-Design Agent

> Trend → moodboard → linesheet → tech-pack · Claude Code skill.


## What it does

Converts trend + brand context into manufacturable design options and auto-generates complete tech packs (specs, BOM, trims, artwork) for vendor-ready handoff. Re-runnable Claude Code skill per category × season × geography.

Proven at Superdry SDX · 120 designs in 5 days with 2 designers. 90% reduction in design time. Live on AJIO.
