01 · The job to be done
The pain
Manual design scaffolding takes weeks. Trend signals from WGSN, Pinterest, retail crawlers arrive unstitched to brand DNA. Tech-pack handoff to vendors loses fidelity — colour, print placement, BOM drift between spec and stitch.
What the agent does
Ingests trend signals + brand context + season constraints. Generates artwork variants, then full garment images (front + back, on-model, 2K). VLM reviewer scores 30+ quality checks per garment with auto-retry on failure. Calculates BOM cost per piece. Produces silhouette and colour variations from reference images. Delivers share-ready line sheets to vendors and buyers.
What humans still do
Buyer reviews finished design packages — not briefs. Designer takes selected concepts into pattern-making and final tech-pack approval. Agent does the production-ready scaffolding; humans select what to range.
02 · How it works · the closed loop
Inputs (blue) → 8-step autonomous pipeline (grey) with VLM quality gate (yellow, auto-retry on fail) → vendor-ready package (green).
01
Brief ingestion
Brand DNA, season direction, category specs, moodboard palette, target price band — pulled from buyer brief and the upstream Category Intelligence report.
02
Context agent
Extracts brand palette, silhouette direction, print techniques and fabric preferences from the brief and reference set.
03
Artwork generation
0-3 artwork images per garment with smart print detection. Solid garments skip artwork; printed garments get placement or AOP (all-over print) variants.
04
Garment image
Full garment image (front + back, on-model, 2K resolution). Zone-aware artwork placement, gender-correct model, age-appropriate styling.
05
Quality review · VLM
VLM reviewer scores 30+ quality checks per garment — colour accuracy, print fidelity, silhouette, category compliance. Auto-retry to step 03 on failure; ship only when checks pass.
06
BOM costing
Cost per piece calculated via Costing Engine API — fabric, trims, SMV (standard minute value), overheads.
07
Variation engine
Generates N silhouette and colour variations from reference images using the moodboard palette. Used for range building and style explosions.
08
Delivery
Images uploaded to cloud storage via presigned URLs. Per-style webhook callbacks fire to downstream systems (PLM, vendor portals) in real time.

05 · Evals · what gates each output

Two layers. Layer 1 — VLM reviewer scores every generated garment against 30+ checks (colour, print, silhouette, category) and auto- retries on failure. Layer 2 — buying team selection rate: % of agent designs taken into the range is the real-world signal. Trend business runs hit ~60% selection.

Gates and thresholds.

  • VLM 30+ checks pass before image ships
  • Selection rate stays above 50% per brand cohort
  • Cost-per-piece estimate within 10% of vendor RFQ on commercial sample
