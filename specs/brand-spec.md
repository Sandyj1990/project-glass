# Brand Specification
## Brand Identity
- Name: Fynd × Reliance Retail
- Project: Project Glass
- Voice: Authoritative, data-driven, frontier-tech, autonomous retail.

## Color Palette
- Ink (Primary Text): `#0A0A0A`
- Ink Muted (Secondary Text): `#6B7280`
- Ink Soft (Tertiary Text): `#9CA3AF`
- Background (Page): `#FFFFFF`
- Background Soft (Section Alternation): `#FAFAFA`
- Border: `#D4D4D4`
- Border Soft: `#F0F0F0`
- Caption: `#4B5563`
- Accent (Primary Brand Color): `#6B5BD6`
- Accent Soft: `#EDE9FE`
- Green (Success/Live): `#16A34A`
- Amber (Warning/Build): `#D97706`
- Red (Error/Alert): `#DC2626`

## Typography
- Primary Font: Inter, system-ui, sans-serif
- Monospace Font: JetBrains Mono, ui-monospace, monospace
- Headings: Display font weights (700) with tight letter-spacing (-0.04em to -0.03em)
- Body Text: Line height 1.55, antialiased.

## Logo Usage
- Use existing `assets/fynd-logo.svg` or similar text-based brand mark.

## Imagery Style
- Real product screenshots
- Architecture diagrams
- Data visualizations (sparklines, charts)
- No stock photography
