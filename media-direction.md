# Media Direction — glass.jiocommerce.io v0.8.4

**Author:** Manus AI
**Purpose:** Master visual direction for all 10 sections of the approved homepage redesign.
**Constraints:** No new claims, exact current colours only (`var(--accent)`, `var(--ink)`, `var(--bg)`, etc.), real UI/assets only, no generic AI art.

---

## 01. Impact
**Visual Purpose:** Establish the massive scale of Reliance Retail and Fynd's operating layer underneath it.
**Primary Image Idea:** A high-altitude, abstract isometric grid representing 19,979 stores connected by glowing data nodes (using `var(--accent)`).
**Backup Image Idea:** A clean, typography-led composition featuring the "380 M+ Customer identities" metric suspended over a subtle glass surface.
**Video Loop Idea:** Slow, continuous data flow along an isometric grid map of India, with nodes pulsing as they process "700 K+ orders / day".
**Composition:** Wide, expansive, breathable. Focus is center-weighted.
**Camera/framing direction:** Top-down isometric or extreme wide angle.
**Lighting:** Studio, soft diffuse overhead.
**Colour usage:** `var(--bg)` background, `var(--ink)` structural lines, `var(--accent)` data pulses.
**Motion direction:** Slow, deliberate, continuous data flow. No sudden flashes.
**Aspect ratios:** Desktop (16:9), Tablet (4:3), Mobile (1:1).
**Export format recommendation:** WebM (VP9) for video, WebP for stills.
**Alt-text guidance:** "Abstract isometric grid showing data flowing between Reliance Retail stores."
**Negative prompt:** No glowing globes, no matrix code rain, no stock photo people, no 3D buildings.
